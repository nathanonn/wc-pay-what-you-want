## Quick Start

1. **Install and activate** the plugin (requires WooCommerce)
2. Go to **WooCommerce > Settings > Pay What You Want** to configure global defaults
3. Edit any product and open the **Pay What You Want** tab to enable it
4. Visit your product page to see the PWYW pricing interface
